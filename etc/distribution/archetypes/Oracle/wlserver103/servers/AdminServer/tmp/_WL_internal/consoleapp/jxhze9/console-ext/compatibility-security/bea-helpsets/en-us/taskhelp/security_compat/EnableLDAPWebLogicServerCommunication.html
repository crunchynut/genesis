Enable communication between the LDAP Server and WebLogic Server


To enable communication between the LDAP server and WebLogic Server:

  1. Expand Compatibility Security > Realms and click the name of the LDAP V1 security realm.
  2. Select Configuration > LDAP Realm V1 and specify values for the attributes used to connect to the LDAP server used by the LDAP V1 security realm.
  3. Click Save to save your changes.
