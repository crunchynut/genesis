Define RDBMS security realm database attributes

Before you begin

Configure an RDBMS security realm. See Configure RDBMS security realms.


To define attributes for the JDBC driver that connects to the database in the RDBMS security realm:

  1. In the left pane, expand Compatibility Security > Realms and click the name of the RDBMS security realm.
  2. Under Configuration > Database tab for the RDBMS security realm, define attributes for the JDBC driver being used to connect to the database.
  3. Click Save to save your changes.
