Compatibility Security: Groups

Related Tasks     Related Topics

A group represents a set of users who usually have something in common, such as working in the same department in a company. Groups are a means of managing a number of users in an efficient manner. When a group is granted a permission in an ACL, all members of the group effectively receive that permission. Oracle recommends assigning permissions to groups rather than to individual users.

This Groups page displays key information about each group that has been configured in this Compatibility security realm.
