Compatibility Security: Access Control Lists

Related Tasks     Related Topics

This page lists displays key information about each ACL that has been configured in this security realm (CompatibilityRealm). This page applies only to WebLogic Server deployments using Compatibility security.

Users access resources in a WebLogic Server security realm. Whether or not a user can access a WebLogic resource is determined by the access control lists (ACLs) for that WebLogic resource. An ACL defines the permissions by which a user can interact with the WebLogic resource. To define ACLs, you create an ACL for a WebLogic resource, specify the permission for the WebLogic resource and then grant the permission to a specified set of users and groups. Oracle recommends assigning ACLs to groups.
