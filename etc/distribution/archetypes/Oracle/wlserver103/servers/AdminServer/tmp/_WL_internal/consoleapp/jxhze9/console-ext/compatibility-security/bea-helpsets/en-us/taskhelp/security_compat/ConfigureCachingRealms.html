Configure Caching realms


To configure the Caching realm:

  1. Configure the alternate or custom security realm with which you will use the Caching realm.

    See the appropriate realm configuration procedures in the following sections:

    • Configure LDAP V1 security realms
    • Configure LDAP V2 security realms
    • Configure Windows NT security realms
    • Configure UNIX security realms
    • Configure RDBMS security realms
    • Configure custom security realms
  2. Expand Compatibility Security > Caching Realms and click New.

    The Create a New Caching Realm page appears.
  3. Enter a name for the Caching realm and click OK.
  4. On the Summary of Security Compatibility: Caching Realms page click the name the new Caching realm.
  5. Select the appropriate configuration tab and enable the caches you want to use with the Caching realm.

    For more information, see:

    • Enable the ACL cache
    • Enable the Authentication cache
    • Enable the Group cache
    • Enable the User cache
    • Enable the Permission cache
  6. When you finish enabling caches for the Caching realm, reboot WebLogic Server.
