Domain: Compatibility Security: Advanced

Configuration Options     Related Topics

This page allows you to define advanced compatibility security settings for this WebLogic Server domain.

Use this page to determine how user, group, and ACL information is returned in a thread and enable the logging of all security checks. This page applies to WebLogic Server deployments using Compatibility security.

Configuration Options

Name Description
Results Batch Size

The number of users, groups, and ACLs to return per Remote Procedure Call (RPC).
Log All Checks

Specifies whether this WebLogic Server domain should log all security checks.
