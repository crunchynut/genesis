Protect user accounts


To protect user accounts, configure password requirements and lockout behavior in your WebLogic Server domain:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane, select your domain (for example, mydomain).
  3. Select Compatibility Security > Passwords.
  4. Define the desired password and lockout values and click Save.

After you finish

Restart WebLogic Server so that your changes can take effect.
