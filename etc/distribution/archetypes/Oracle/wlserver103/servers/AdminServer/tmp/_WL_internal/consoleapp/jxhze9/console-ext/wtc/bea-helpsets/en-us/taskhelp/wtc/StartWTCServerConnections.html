Start WTC server connections


To start the connection that will be used with this WTC Server:

  1. In the Administration Console, expand Interoperability and select WTC Servers.
  2. On the WTC Servers page, click the name of a WTC Server.
  3. Click the Control tab and then click the Connections tab.
  4. On the Connections page, check a connection with a Connected column value as false.
  5. Click Start to start the connection. The Connected column value should change to true.
