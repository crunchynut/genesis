Compatibility Security: Users

Related Tasks     Related Topics

Users are entities that can be authenticated in a WebLogic Server security realm. A user can be a person or a software entity, such as a Java client. Each user is given a unique identity within a WebLogic Server security realm.

This page allows you to create a new user, change an existing user's password, delete a user, and unlock users in a manageable security realm (for example, the File realm). The current list of users is shown in the table below.
